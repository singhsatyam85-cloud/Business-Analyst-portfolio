# Discovery Questions

## A. Business Objective Questions
1. What is the main reason ABC School wants this application?
2. What problem are you facing with the current attendance process?
3. Who will approve the final requirements from ABC School side?
4. Is the first version focused only on attendance and basic admin work?
5. What result will make this project successful for ABC School?

## B. Current Attendance Process Questions
1. How do teachers currently mark attendance?
2. Is attendance currently recorded on paper, Excel or any existing system?
3. How many classes are there in the school?
4. How many students are usually in one class?
5. Who checks attendance after teachers submit it?

## C. Teacher Mobile App Questions
1. Should teachers log in using username and password?
2. Should teachers see only their assigned class?
3. What attendance options should be available? Example: Present, Absent, Late.
4. Should the teacher review attendance before submitting?
5. After submission, should the teacher be able to edit attendance?

## D. Admin Web Application Questions
1. Who from the admin team will use the web application?
2. What student details does admin need to add?
3. What teacher details does admin need to add?
4. Should admin be able to update student and teacher records?
5. Should admin view attendance by class, date and student?

## E. Role-Based Access Questions
1. What should Admin be allowed to access?
2. What should Teacher be allowed to access?
3. Should teachers be blocked from admin screens?
4. Should admin be able to view all attendance records?
5. Should each user have a unique login?

## F. Reporting Questions
1. What basic attendance reports are needed?
2. Should reports be filtered by class?
3. Should reports be filtered by date?
4. Should reports show individual student attendance?
5. Should reports be downloadable in Project 1, or only viewable on screen?

## G. Out-of-Scope Confirmation Questions
1. Can we confirm fee management is out of scope for Project 1?
2. Can we confirm transport management is out of scope?
3. Can we confirm exam marks are out of scope?
4. Can we confirm parent login is out of scope?
5. Can we confirm payment gateway and notifications are out of scope?
