# ABC School Web and Mobile Application

This is a sample Business Analyst portfolio project for a school web and mobile application.

## Project Purpose

ABC School wants to reduce manual attendance work and improve basic student, teacher and attendance management.

## Scope

The project includes:

- Teacher login
- Teacher attendance marking
- Attendance submission
- Admin student management
- Admin teacher management
- Basic attendance reports
- Role-based access
