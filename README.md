## Role Fit Assistant Setup

The portfolio includes a static **Role Fit Assistant** called **Recruiter Fit Checker**.

Recommended repository folder:

```text
cv-data/shiv-satyam-singh-detailed-cv.txt
```

Add your detailed CV/profile text in this file. The assistant will try to read this text file from the GitHub Pages repository and compare pasted job descriptions against it.

Important notes:

- Use a `.txt` file for the detailed CV/profile text.
- Do not use private laptop folders, because GitHub Pages cannot read them.
- Do not rely on PDF or Word files for this matching feature, because browser-based static pages cannot reliably extract text from those files without extra libraries.
- No backend, paid service, or external data collection is used.
- If the detailed text file is not found, the page uses a built-in portfolio summary as a fallback.
