Teacher-Led STEM and Coding Learning Platform

Business Analyst Portfolio Case Study

Project Overview

This is a self-initiated Business Analyst case study for a teacher-led web platform. The platform allows teachers to open STEM and coding learning content on a classroom screen, projector or smart board.

Business Problem

Teachers need a simple way to explain difficult STEM and coding concepts using visual, animated and interactive classroom-friendly content. The current process depends on manual searching, scattered files and inconsistent content access.

Proposed Solution

A web-based teacher portal where teachers can search, filter and open approved STEM content during classroom sessions. Admin users can upload, review and manage learning content.

Sample User Story

As a teacher, I want to search STEM content by subject, class and topic, so that I can quickly open the right learning material during a classroom session.

Sample Acceptance Criteria

  • Given the teacher is logged in, when they select a subject, class and topic, then matching content should be displayed.
  • Given no matching content is available, when the teacher searches, then the system should show a clear no-results message.
  • Given content is selected, when the teacher clicks open, then the content should open in classroom display mode.
