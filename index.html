Web & Mobile App BA Project Work | FeelTech Digivation Pvt Ltd

Back to Project Work

Project Background

A school-focused attendance management system designed to support teacher and admin workflows for recording attendance and improving visibility of attendance records.

Business Problem

Attendance tracking needed clearer digital workflows, better record visibility, and reduced dependency on manual follow-up between teachers and admin users.

My BA Role

Supported requirement gathering, workflow analysis, user story writing, acceptance criteria preparation, process mapping, UAT support, and feedback tracking.

Key Users / Stakeholders

  • Teachers
  • School admin users
  • School leadership
  • Implementation and support team

Key Features or Scope

  • Teacher login flow
  • Class and date selection

  • Basic attendance reporting
  • Role-based access

BA Deliverables Created

  • User stories
  • Acceptance criteria
  • Process flow
  • UAT scenarios
  • Feedback tracker

Tools Used

Jira, Confluence, Microsoft Excel, Microsoft PowerPoint, Draw.io or Visio-style process mapping, Agile Scrum working practices.

Outcome / Impact

The work helped clarify the teacher and admin journey, define the MVP workflow, support testing conversations, and make attendance records easier to manage.
